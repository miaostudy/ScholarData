# Api
## google scholor
**有免费套餐，但是不多**

输入：作者名

输出：所有的论文标题、部分摘要、发表信息。

## Aminor
## 根据名字获取期刊、作者、论文id
**免费**

输入：名字

输出：id

### 学者论文
**1.5一次**

输入：学者id

输出：他的所有论文id和标题

### 论文详情
**0.01一次**

输入：论文id

输出：摘要、关键字等，**无正文**

### 论文批量查询

**0.1一次**

输入：关键词数组

输出：具有这些关键词的论文信息

 
### 根据条件获取论文详情

**0.2一次**

输入：期刊id和年份

输出：所有论文的摘要、关键词、作者等信息

### 混合搜索

**0.05一次**

输入：主题查询、作者查询、组织查询均可

输出：论文id

单用作者名查询非常容易搜不到，Kaiming He等人都搜不到,但是用1.5的那个接口就能获取到
